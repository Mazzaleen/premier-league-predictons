# Premier League Predictions

Annual group-chat prediction game. Each season's submissions become a self-contained HTML infographic, scored against actual results.

Live: https://mazzaleen.github.io/premier-league-predictons/ · Rubric: [`scoring.html`](scoring.html)

Static site — no build step. Open `index.html` or run `python3 -m http.server`. Pushes to `main` auto-deploy via GitHub Pages.
